This guide takes you from zero to a working authenticated request in under 5 minutes.

## Prerequisites

- A terminal with `curl`, or a Node.js / Python environment.
- An email address you can receive mail at (for the OTP code).

## The flow

<Callout kind="info">
  Flow: register a user, verify the OTP, then make an authenticated request.
</Callout>

<Steps>
  <Step title="Register a new account">
    POST to `/users/register/` to create the account and send a 6-digit OTP to the email address.
  </Step>
  <Step title="Verify the OTP">
    POST to `/users/verify-otp/` with the email and OTP code to receive access and refresh tokens.
  </Step>
  <Step title="Call a protected endpoint">
    Use the access token in the `Authorization: Bearer <token>` header to request `/users/me/`.
  </Step>
</Steps>

## Steps

<Steps>
  <Step title="Register a new account">
    Send the email and password. The API replies with a 6-digit OTP sent to that email.

    <Tabs>
      <Tab title="curl">
        ```bash
        curl -X POST 'https://api.ibtiaath.com/api/v1/users/register/' \
          -H 'Content-Type: application/json' \
          -d '{
            "email": "user@example.com",
            "password": "SecurePass123!"
          }'
        ```
      </Tab>
      <Tab title="JavaScript">
        ```js
        import axios from 'axios';

        const api = axios.create({ baseURL: 'https://api.ibtiaath.com/api/v1' });

        const { data } = await api.post('/users/register/', {
          email: 'user@example.com',
          password: 'SecurePass123!',
        });
        console.log(data); // { message: "Verification code sent to your email", email: "..." }
        ```
      </Tab>
      <Tab title="Python">
        ```python
        import requests

        BASE = "https://api.ibtiaath.com/api/v1"

        r = requests.post(f"{BASE}/users/register/", json={
            "email": "user@example.com",
            "password": "SecurePass123!",
        })
        r.raise_for_status()
        print(r.json())
        ```
      </Tab>
      <Tab title="Dart">
        ```dart
        import 'package:http/http.dart' as http;
        import 'dart:convert';

        const base = 'https://api.ibtiaath.com/api/v1';

        final r = await http.post(
          Uri.parse('$base/users/register/'),
          headers: {'Content-Type': 'application/json'},
          body: jsonEncode({
            'email': 'user@example.com',
            'password': 'SecurePass123!',
          }),
        );
        print(jsonDecode(r.body));
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Verify the OTP">
    Check the email for a 6-digit code, then verify it. The response contains the access and refresh tokens.

    <Tabs>
      <Tab title="curl">
        ```bash
        curl -X POST 'https://api.ibtiaath.com/api/v1/users/verify-otp/' \
          -H 'Content-Type: application/json' \
          -d '{
            "email": "user@example.com",
            "otp_code": "123456"
          }'
        ```
      </Tab>
      <Tab title="JavaScript">
        ```js
        const { data: tokens } = await api.post('/users/verify-otp/', {
          email: 'user@example.com',
          otp_code: '123456',
        });
        const { access, refresh, user } = tokens;
        ```
      </Tab>
      <Tab title="Python">
        ```python
        r = requests.post(f"{BASE}/users/verify-otp/", json={
            "email": "user@example.com",
            "otp_code": "123456",
        })
        tokens = r.json()
        access, refresh = tokens["access"], tokens["refresh"]
        ```
      </Tab>
    </Tabs>

    Example response:

    ```json
    {
      "message": "Email verified successfully",
      "access": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
      "refresh": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
      "user": {
        "id": 42,
        "email": "user@example.com",
        "role": "user",
        "is_email_verified": true,
        "profile_completed": false
      }
    }
    ```

    <Callout type="tip">
      Save the `refresh` token in secure storage (Keychain on iOS, Keystore on Android, HTTP-only cookie on web). The `access` token expires in 1 hour; the `refresh` token in 7 days.
    </Callout>
  </Step>

  <Step title="Make an authenticated request">
    Send the access token as a Bearer header.

    <Tabs>
      <Tab title="curl">
        ```bash
        curl -X GET 'https://api.ibtiaath.com/api/v1/users/me/' \
          -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
        ```
      </Tab>
      <Tab title="JavaScript">
        ```js
        api.defaults.headers.common['Authorization'] = `Bearer ${access}`;
        const { data: me } = await api.get('/users/me/');
        ```
      </Tab>
      <Tab title="Python">
        ```python
        r = requests.get(
            f"{BASE}/users/me/",
            headers={"Authorization": f"Bearer {access}"},
        )
        me = r.json()
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Refresh the token before it expires">
    Access tokens last 1 hour. Use the refresh token to get a new pair before it expires.

    ```bash
    curl -X POST 'https://api.ibtiaath.com/api/v1/study-abroad/auth/refresh/' \
      -H 'Content-Type: application/json' \
      -d '{ "refresh": "eyJ..." }'
    ```

    The response is a new `{ access, refresh }` pair. The old refresh token is blacklisted immediately — store the new one.
  </Step>
</Steps>
