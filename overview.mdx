The EBTE3ATH API is a REST service that powers the Ibtiaath platform — study-abroad applications, flight booking, eSIM purchase, document translation, gift codes, and wallet payments.

```http
Base URL: https://api.ibtiaath.com/api/v1/
```

<Callout type="warning">
  Every URL ends with a trailing slash. Requests without one return a 301 redirect.
</Callout>

## Conventions

| | |
|---|---|
| **Auth** | `Authorization: Bearer <jwt-access-token>` |
| **Format** | All requests/responses are JSON |
| **Currency** | IQD (Iraqi Dinar) unless stated |
| **Timezone** | `Asia/Baghdad` |
| **Pagination** | `?page=1&page_size=20` — see [Pagination](/pagination) |
| **Rate limits** | See [Rate limits](/rate-limits) |
| **Errors** | Standard HTTP codes + structured JSON — see [Errors](/errors) |
