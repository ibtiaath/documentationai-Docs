The Ibtiaath API is a REST service that powers the Ibtiaath platform. It exposes endpoints for study-abroad applications, flight bookings, eSIM purchases, document translations, gift codes, and Wayl payments.

## Base URL

```
https://api.ibtiaath.com/api/v1/
```

Every URL ends with a trailing slash. Requests without one return a 301 redirect.

## Authentication

All authenticated endpoints require a JWT bearer token:

```
Authorization: Bearer <access_token>
```

Access tokens expire after 1 hour. Refresh tokens last 7 days. See the API Reference for the auth endpoints.

## Conventions

- All requests and responses are JSON.
- Currency is **IQD** (Iraqi Dinar) unless stated otherwise.
- Timestamps are ISO 8601 in `Asia/Baghdad` timezone.
- Pagination uses `?page=N&page_size=20`.

## Modules

The API is organized into the following modules:

- **Users** — registration, authentication, profile
- **Study Abroad** — programs, scholarships, applications
- **Flights** — search and booking via Duffel
- **eSIM** — data plans via Airalo
- **Translations** — certified document translation orders
- **Gift Codes** — wallet top-up via prepaid codes
- **Wayl Payments** — card payments via the Wayl gateway
- **Notifications** — FCM push token management
